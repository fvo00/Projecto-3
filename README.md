# Proyecto Laravel Hub - ASIX 2026

Este proyecto es una aplicación web interactiva desarrollada con **React** y **Vite** que sirve como guía técnica para el framework Laravel.

##  Instalación
1. Clona el repositorio:
   `git clone https://github.com/fvo00/Projecto-3.git`
2. Instala las dependencias:
   `npm install`
3. Ejecuta el proyecto:
   `npm run dev`

## Scripts
- `npm run dev`: Inicia el servidor de desarrollo.
- `npm run build`: Crea la versión optimizada para despliegue.

## Estructura del Proyecto
- `/src/components`: Componentes reutilizables (Card, Footer.).
- `/src/pages`: Vistas principales de la aplicación.
- `/src/services`: Funciones para obtener datos de los JSON.
- `/public/data`: Archivos JSON que simulan la base de datos.

## Notas
Proyecto realizado para la asignatura de Entornos de Aplicación Web (IAW).
